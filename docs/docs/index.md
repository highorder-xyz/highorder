# HighOrder和Hola

Hola 是 HighOrder Language 的缩写，也称为高阶语言。是一种面向产品经理，开发，企业IT，业务人员在内的快速开发软件系统的语言。而HighOrder是运行Hola语言的运行环境，也就是说Hola编写的程序放到HighOrder上运行，就能展示出相应的界面和功能。
